# customer-retention-cohort-analysis
SQL-based cohort analysis and retention dashboard

## 📊 Project Overview
This project analyzes customer retention using cohort analysis.

## 🛠 Tools
- PostgreSQL (DBeaver)
- Looker Studio

## 📌 Steps
- Extracted first purchase date per customer
- Created cohort groups
- Calculated retention rates
- Visualized results in a dashboard

## 📈 Key Insights
- Retention drops significantly after the first month
- Stabilizes around 15–20% in later months
